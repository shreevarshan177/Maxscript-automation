# MaxScript Animation Automation

## Overview
This project is a custom MaxScript tool developed to automate repetitive animation tasks in Autodesk 3ds Max.

## Problem
Creating animations manually required performing repetitive operations, which increased the time required to complete tasks.

## Solution
The script automates these repetitive operations, helping improve workflow efficiency and reduce manual effort.

## Technologies Used
- MaxScript
- Autodesk 3ds Max

## Features
- Automates repetitive animation tasks
- Reduces manual work
- Improves consistency
- Easy to execute within 3ds Max
